I spend a lot of my time building and contributing to open-source, mostly in Go. It’s where I can experiment, refine ideas, and focus on writing code that feels both efficient and clean to use.

Lately I’ve been exploring more around web and infrastructure, with a growing interest in developer experience and how to build tools that are both powerful and simple to use 🚀

I’m especially into problems where the details matter and where good design makes a real difference.
